# Yakr Protocol — name, trademark, and independence

**Status:** Informational (not legal advice)  
**Last UK IPO search:** 8 July 2026

The public name for this project is **Yakr Protocol** — an open messaging protocol (from *yakking*, slang for talking). The short form **Yakr** refers to the protocol in context (`yakr-v1.0`, reference code packages). **Yakr Protocol Certified** (short: **Yakr Certified**) is the conformance program in [CERTIFICATION.md](CERTIFICATION.md).

**Conclusion (UK, informational):** Reasonable clearance to use **Yakr Protocol** for an open messaging specification and steward activities, with clear branding and the disclaimers below. This is not a formal freedom-to-operate opinion.

## Not affiliated — sound-alike businesses

**Yakr Protocol** is independent of unrelated companies and marks that sound or look similar, including:

- **YAKR Pty Ltd** (Australia) — international recruitment and migration ([yakr.com.au](https://yakr.com.au/)); different industry and product
- **Yakkr Limited** (United Kingdom) — software / streamer growth ([yakkrgrowth.com](https://yakkrgrowth.com/))
- **Yakry** — registered UK class 9 mark used for electronic hardware (e.g. vehicle backup cameras)

No endorsement, partnership, or affiliation with these entities is implied. If you operate a similarly named business and believe users could be confused, contact the repository steward via a GitHub issue labeled `certification`.

## Trademark registration

- **Registration is not required** to publish the spec or use **Yakr Protocol** in docs, provided use does not infringe third-party rights.
- The steward may file a UK or other trade mark later for **Yakr Protocol** / certification badges; status will be updated here if filed.
- Third parties must not use **Yakr Protocol Certified** / **Yakr Certified** badge artwork or imply official endorsement without a certification agreement ([CERTIFICATION.md](CERTIFICATION.md)).

## Permitted use without certification

- Factual statements: “implements the **Yakr Protocol**”, “based on `yakr-v1.0`”
- Links to this repository and the normative spec
- Independent product names that do not include **Yakr Certified** or confusingly similar marks

## Planned web presence (domains not yet registered by steward)

When registered, canonical URLs are intended to be:

Domain registration does not by itself establish trade mark rights; it supports a clear public home for **Yakr Protocol** separate from sound-alike businesses.
