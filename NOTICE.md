# Trademark and project name

**Status:** Informational (not legal advice)

The name **Yakr** and associated **Yakr Certified** badge artwork are used by this project to identify the open messaging protocol and conformance program described in [CERTIFICATION.md](CERTIFICATION.md).

## Current status

- The protocol specification and reference implementation in this repository are open source (license TBD in [README.md](README.md)).
- **Trademark registration** for “Yakr” in connection with messaging software and certification services is **planned**; status will be updated here when filed or granted.
- Until registration is complete, third parties should not use **Yakr Certified** branding or imply official endorsement without a certification agreement.

## Permitted use without certification

- Factual statements: “implements the Yakr protocol”, “based on yakr-v1.0”
- Links to this repository and the normative spec
- Independent product names that do not include “Yakr Certified” or confusingly similar marks

## Questions

Certification and trademark licensing inquiries: open a GitHub issue labeled `certification` (process TBD) or contact the repository owner.
