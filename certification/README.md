# Yakr Protocol Certification

**Status:** Open — accepting applications (baseline `yakr-v1.0`, Phase 11 complete)

Official program: [CERTIFICATION.md](../CERTIFICATION.md)

## Apply

1. Run the self-test bundle:

   ```bash
   ./scripts/certification_self_test.sh
   ```

2. Complete the checklist for your category:

   - [Client v1.0](client-checklist.md)
   - [Relay v1.0](relay-checklist.md)

3. Open a GitHub issue using **Yakr Certification Application** (label `certification`), or email the steward with the same fields if you cannot use GitHub.

4. After review: certificate ID, badge assets under `certification/badge/`, and listing in [IMPLEMENTERS.md](IMPLEMENTERS.md).

## Fees (published)

| Tier | Fee |
|------|-----|
| Open source / nonprofit | **Free** (listing after review) |
| Commercial client | **£500** one-time + **£200/year** renewal |
| Commercial relay | **£300** one-time + **£150/year** renewal |
| Major version re-certification | **£150** delta review |

The normative spec and test vectors remain free. Fees fund conformance review and steward maintenance only.

## Badge use

Requires an approved application. See [CERTIFICATION.md § Badge and trademark](../CERTIFICATION.md#badge-and-trademark).

**Every certified badge use must include** (or link to) this disclaimer:

> Certification indicates interoperability and protocol-conformance review only. It is not a security audit, product endorsement, or assurance of production suitability.

Allowed without certification: factual “implements Yakr protocol v1.0” wording and links to this repository.
