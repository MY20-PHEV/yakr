Worked Example: Mailbox Tag

From docs/spec/test-vectors-v1/mailbox_tag.json:

Inputstest vector
master_secret = aa aa aa ... (32 bytes)
direction     = "alice->bob"
epoch         = 1000
epoch_secs    = 3600
Step 1 — mailbox secret
mailbox_secret = HKDF-SHA256(
  ikm  = master_secret,
  salt = "",
  info = "yakr/v0.1/mailbox-tag" || "alice->bob"
)
Step 2 — tag
tag = HMAC-SHA256(
  key = mailbox_secret,
  msg = "alice->bob|1000"
)

expected = 7f409c7b34bd6c82131f8fbb1082a433
           94f045b78c004c320bb6f347f51acf08
Tags rotate every epoch_secs (default 3600 s). Fetch checks current epoch plus lookback epochs to handle clock skew and delayed delivery.

PQ Rekey Policy

Hybrid sessions must rekey when either threshold is reached:

flowchart TD
  H{Hybrid session?} -->|no| OK[Continue]
  H -->|yes| C{messages ≥ 10,000
OR age ≥ 7 days?}
  C -->|yes| RK[YAKR_ERR_PQ_REKEY_REQUIRED]
  C -->|no| OK
