Complete Message Example

Alice sends "hello vector" to Bob (seq=1). This is the full stack from plaintext to relay store.

flowchart TB
  A1["① Inner JSON
body: hello vector"] --> A2["② Encrypt
message_key(1)"]
  A2 --> A3["③ Outer blob
tag + expiry + ciphertext"]
  A3 --> A4["④ POST /v1/blobs"]

① Inner Message (plaintext JSON)

Source: test-vectors-v1/inner_message.json

inner-text-v1before encryption
{
  "body": "hello vector",
  "conversation_id": "pairwise_alice_bob",
  "created_at": 1700000000000,
  "message_id": null,
  "sender_device_id": "abc123",
  "seq": 1,
  "type": "text",
  "version": 1
}

Keys are sorted lexicographically on the wire. After encryption the result becomes the outer blob's ciphertext field.

CLI Demo Commands

Docker — full offline demo
./scripts/demo_offline_delivery.sh      # Phase 1 single-hop
./scripts/demo_two_hop_delivery.sh      # Phase 2 onion + receipts
./scripts/demo_invite_pairing.sh        # Phase 4 invites
./scripts/demo_profile_delivery.sh      # Phase 5 profiles
./scripts/demo_hybrid_pairing.sh        # Phase 6 hybrid PQ
./scripts/demo_relay_group_pairing.sh   # Relay rendezvous (local)
./scripts/demo_vps_charlie_relay.sh     # Alice/Bob Docker + VPS Charlie
Local dev
# Terminal 1
uv run yakr-relay serve --port 8080 --data-dir ./data/relay

# Terminal 2 — Alice
uv run yakr --name alice send bob "hello world"

# Terminal 3 — Bob (can be offline during send)
uv run yakr --name bob fetch alice

Verify Your Implementation

Interop tests
uv sync --all-packages
uv run pytest packages/yakr-testkit/tests/test_phase9_interop.py -q
uv run pytest packages/yakr-testkit/tests/test_phase9_relay_abuse.py -q
Independent implementations should pass the interop verifier using only docs/spec/yakr-protocol-v1.md and docs/spec/test-vectors-v1/ — no yakr_core imports required.
